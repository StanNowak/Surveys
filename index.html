üìä Features

  • Background Questionnaire - 3 quick demographic questions
  • Question blocks - Avalanche problems comprehension
  • Individual Questions - General knowledge or disambiguation of avalanche problems
  • Auto-advance - Oprions for multiple or one question per page with automatic progression
  • Randomization - Randomization of questions and question blocks (potential for more advanced implementations)
  • Timing Capture - Per-question response times and idle tracking (navigating off screen)
  • Immediate Feedback - Graded results with explanations (for a good learning experience)
  • Data Download - JSON export of all responses and timing (dummy data that can be fed to a backend)
  • ID management - UUID for each participant, URL parameters for customization

üîß Configuration

The survey supports URL parameters for customization:

  • uuid - Unique participant identifier
  • bank - Item bank selection
  • config - Configuration file selection

Example:
./public/?uuid=participant-123&bank=full

üìÅ Survey Content

  • Avalanche Problem Blocks - 8 avalanche problem scenarios with 4 questions each
  • Individual Diagnostic Questions - 3 general knowledge assessment questions
  • Background Questions - 3 demographic and experience questions

üíæ Data Output

Survey completion automatically downloads a JSON file containing:

  • All survey responses
  • Per-question response times (rt_questionname_final)
  • Per-question idle times (idle_questionname_ms)
  • Total idle time (idle_ms)
  • Grading results with correct/incorrect answers
  • Participant UUID and timestamps
